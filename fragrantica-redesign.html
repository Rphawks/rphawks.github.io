PROJECT BRIEF

The goal is to redesign a simple website. One should take away the skills necessary to analyze and identify flaws in an existing interface, create low-fidelity and high-fidelity prototypes for various screen sizes, and build a responsive website based on those prototypes.

WEBSITE CHOICE & ANALYSIS
live original site can be found at
fragrantica.com

Fragrantica is a bit like Wikipedia for fragrances, and it gets very relatively high traffic for a website that has an incredibly cluttered and dated UI because it has data that others do not.

Learnability: While the page is quite cluttered, most basic tasks are clearly marked, as long as one is able to find the section on the page where the task is done.

Efficiency: The interface is rather low efficiency, as the clutter increases the time to find the actionable section. The layout of the interface is sequential and therefore each section is not particularly memorable, meaning that actions must be confirmed visually instead of being assisted by various spatial cues.

Memorability: As mentioned above, the interface has memorable portions but the sequential layout affects the ability to complete the same task multiple times without additional search effort.
