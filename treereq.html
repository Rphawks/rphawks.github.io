PROJECT BRIEF

Over the course of the semester, through assignments, various features of a fullstack hypertext system (nodes, anchors, links, editable nodes, etc.) were built out using the MERN stack.

The final project builds on top of this developed architecture by

1. adding functionality for at least one additional node type with a new user interface
2. adding at least one global feature (trails, access rights, visualisations, etc.) that works across all nodes, anchors, and/or links which helps users navigate and/or visualize the structure of the hypermedia system.

TreeReq is a structured course planner and prerequisite/postrequisite visualizer that enables high-level planning not directly accessible through single-course visualization.

TreeReq combines API-retrieved and authorized scraped data from course databases
(Banner, Courses @ Brown, The Critical Review) to generate trees of course interaction and populate user-modifiable course nodes with existing descriptions and review information.
